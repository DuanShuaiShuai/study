# vscode
## 
- 文件资源管理器 ctrl+shift+e
- 跨文件搜索 ctrl+shift+f
- 源代码管理 ctrl+shift+g
- 启动与调试 ctrl+shift+d
- 管理与扩展 ctrl+shift+x
- 查看错误与警告 ctrl+shift+m
- 查看并运行所有的命 ctrl+shift+p
    - Interactive Editor Playground 交互式演习场
- ctrl+p 快速定位文件
- 快速创建新的vscode编辑器 ctrl+shift+n
- 关闭当前的vscode编辑器 ctrl+shift+w
- 打开快捷键列表 ctrl+k
- 关闭当前tab ctr+f4
- code -r -g package.json:128 打开并跳转至128行
- code -r -d a.txt b.text  比较两个文件
